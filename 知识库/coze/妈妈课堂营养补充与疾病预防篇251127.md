# 什么是粘液便

![/private/var/mobile/Containers/Data/Application/B05FB7C3-005A-4958-AF61-0B9B5CD5DB84/tmp/insert_image_tmp_dir/2018-07-08 10:32:40.458000.png2018-07-08 10:32:40.458000](图片占位符6.jpg)
黏液是肠道粘膜的分泌物，起屏障作用，防止异物进入肠粘膜细胞，并润滑肠道，便于大便的排出
当黏液大量分泌时，进入大便中会出现明显的果冻样大便，如果仅出现数次，宝宝的饮食、精神方面都正常，没有发烧、腹痛、腹胀的现象就是肠道受到一过性轻度刺激的反应，属于身体自行调节，无需担心。
如果黏液便中带血、有脓液、腥臭味、腹痛、腹胀、频繁呕吐就是严重肠道疾病的表现。

# 什么是奶瓣
奶瓣是宝宝大便中白色的颗粒状或小块状、瓣状的东西。奶瓣大多是酪蛋白的凝结物。一般肠道功能受损的时候会吸收不完全就跟着大便排出来了，所以宝宝排稀便的时候更容易出现奶瓣。
配方奶中蛋白质含量高尤其是酪蛋白含量高，容易出现奶瓣也是正常的。
只要宝宝喂养正常，体重增长正常，无需去医院检查。

# 泡沫样便与乳糖不耐受症

恶心、呕吐、吐奶
